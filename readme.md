# 🐍 Ejercicios de Lógica y Programación en Python

## 📖 Descripción General

Este repositorio recopila una serie de ejercicios prácticos desarrollados en **Python**, enfocados en reforzar los fundamentos de programación.

El proyecto abarca distintos paradigmas y técnicas esenciales, incluyendo:

- Programación estructurada  
- Programación funcional  
- Manejo de excepciones  
- Programación orientada a objetos  

El objetivo principal es consolidar el pensamiento lógico y la capacidad de resolución de problemas mediante la implementación progresiva de distintos retos.

---

## 🚀 Contenidos Trabajados

### 🔹 Fundamentos de Python

- Definición y uso de funciones  
- Manejo de parámetros y valores por defecto  
- Estructuras condicionales (`if`, `elif`, `else`)  
- Bucles y lógica iterativa  
- Conversión y validación de tipos de datos  

---

### 🔹 Estructuras de Datos

- Listas  
- Tuplas  
- Diccionarios  
- Conjuntos (`set`)  
- Manipulación de cadenas de texto  

---

### 🔹 Programación Funcional

- Funciones `lambda`  
- `map()`  
- `filter()`  
- `reduce()` (módulo `functools`)  
- Uso de funciones de orden superior  

---

### 🔹 Manejo de Excepciones

- Bloques `try / except`  
- Creación de excepciones personalizadas  
- Validación de entradas del usuario  
- Control de errores en tiempo de ejecución  

---

### 🔹 Recursividad

- Implementación de funciones recursivas  
  - Ejemplo: cálculo del factorial  

---

### 🔹 Programación Orientada a Objetos (POO)

- Definición de clases  
- Atributos y métodos  
- Encapsulamiento  
- Simulación de casos prácticos:
  - Sistema bancario  
  - Estructura de árbol  

---

## 🧩 Implementaciones Prácticas

El proyecto incluye ejercicios aplicados a escenarios reales como:

- Procesamiento y análisis de texto  
- Simulación de operaciones bancarias  
- Transformación y filtrado de datos  
- Cálculos matemáticos y agregaciones  
- Sistemas de clasificación (notas, horarios)  
- Aplicación de descuentos en compras
